# v1.1.2 更新日志

## 🚀 重大更新 (Major Updates)
- **多任务并发下载**：
  - 终于来了！解除了强制单线程限制，现在支持**自定义并发数量** (1-10)。
  - 您可以在下载管理界面根据网络状况和账号风险自由调整下载速度。
  - *提示：并发数越高，触发验证码的几率越大，请按需调整。*
- **全局验证码熔断机制**：
  - 更加智能的风控保护：当任意任务触发验证码时，系统会自动**暂停所有正在运行的任务**，防止连续触发导致封控。
  - 验证完成后，支持一键恢复所有暂停的任务，无需逐个点击，体验更流畅。
- **在线常见问题 (FAQ)**：
  - 新增“常见问题”按钮，内容**实时从 GitHub 拉取**，无需更新软件即可获取最新解答。
  - 针对国内网络环境优化，内置 CDN 加速，加载飞快。

## ✨ 界面与体验优化 (UI/UX)
- **主界面升级**：
