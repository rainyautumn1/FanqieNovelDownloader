# 常见问题 (FAQ)

## 🔴 核心问题

### 1. 下载内容不全 / 只有前几章？
- **原因**：部分章节（尤其是VIP章节）需要登录才能阅读。
- **解决**：请在软件右侧的**内置浏览器**中点击右上角“登录”，登录您的番茄小说账号（建议使用 SVIP 账号）。软件会自动同步登录状态并获取下载权限。

### 2. 为什么会出现验证码 / 下载暂停？
- **原因**：这是番茄小说的反爬虫机制（WAF/风控）。
- **解决**：
  1. 软件会自动检测并暂停下载，弹出提示框。
  2. 请在右侧浏览器中手动完成滑块验证或点击验证。
  3. 验证通过后，点击弹窗上的“**验证已完成**”按钮，下载将自动继续。

### 3. 为什么下载速度慢 / 只能同时下载几本？
- **原因**：为了防止账号被封禁，我们限制了并发数量。
- **提示**：您可以在“下载管理”界面调整并发数（1-10），但请注意：**并发数越高，触发验证码的概率越高**。推荐保持默认或低并发。

## 🟡 环境与运行

### 1. 没有 Python 环境怎么运行？
- **Windows 用户**：请前往 GitHub Releases 页面下载 `.exe` 格式的绿色免安装版，解压即用。
- **小白用户**：可以去 Bilibili 搜索“Python安装教程”进行学习，非常简单。

### 2. 支持 Mac / Linux / Android 吗？
- **Mac/Linux**：支持！请下载源码，安装 Python 3.8+ 和依赖包 (`pip install -r requirements.txt`) 后运行 `main.py` 即可。
- **Android/iOS**：暂不支持手机端。

### 3. 浏览器无法打开新标签页？
- **说明**：为了保证下载功能的稳定性，我们强制在当前窗口打开所有链接，这是正常现象。

### 4. 登录状态总是丢失？
- **解决**：最新版本已修复此问题。软件会在 `browser_data` 目录保存您的登录凭证，请**不要删除**该文件夹。
